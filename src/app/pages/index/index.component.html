Purpose

This application is a Twitter copycat application and was created for academic purposes.
The code for this single page application can be found in this repository.
The code for the node.js backend including the server-rendered ui can be found here.

Technologies

This application was build upon a node.js backend.
The used server framework is Hapi.
The server-rendered ui is build with handlebars.
For styling the css framework Semantic UI was used.
This single page application is based on Angular.
The data is saved into the document database mongodb.

Hosting

The node.js backend together with the server-rendered ui is hosted at Heroku.
The single page application frontend is hosted on a private server.
The mongodb database is hosted by mlab.
